zg2g::Range class

Holds a range of some numeric type.

Contents

  • Reference
    • Constructors, destructors, conversion operators
    • Public functions

Holds a range of either integers or real numbers, and produces a random number from the range on demand. The template argument should be of an arithmetic type (either integer or real types).

Parameters

Ranges can hold a single number, which works as if it was just a number. Note: Ranges can hold one number only if it is of integral type!
